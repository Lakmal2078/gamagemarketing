# Gamage Marketing | Premium Digital Agency 🚀

A modern, high-conversion landing page designed for a Digital Marketing Agency in Sri Lanka. Built with a focus on premium UI/UX, smooth animations, and high performance.

## ✨ Features

* **Modern UI & Animations:** Animated background gradients and scroll-triggered element animations using Intersection Observer.
* **Responsive Design:** Fully mobile-friendly layout optimized for all screen sizes.
* **Interactive Modals:** Detailed popup windows for service descriptions without leaving the page.
* **Dynamic Testimonials:** Horizontal scrolling review cards with interactive progress bars and rating summaries.
* **Working Contact Form:** Integrated with Formspree for real-time email submissions.
* **Quick Contact:** Floating WhatsApp button for instant client communication.

## 🛠️ Technologies Used

* **HTML5** (Semantic structure)
* **CSS3** (Custom styles, Flexbox/Grid, Keyframe Animations)
* **JavaScript** (Vanilla JS for DOM manipulation, Modals, and Scroll Observers)
* **Tailwind CSS** (via CDN for quick utility classes)
* **Font Awesome** (For scalable vector icons)

## 🚀 Getting Started

To view this project locally, simply clone the repository and open the `index.html` file in your web browser.

### Prerequisites
No special software is required. A modern web browser (Chrome, Firefox, Safari) is sufficient.
